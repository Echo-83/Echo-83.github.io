# Images Directory

Place your profile photo and other images here.

Example:
- `profile.jpg` or `profile.png` - Your profile photo for the main page
- Any other images you want to use on your course pages

## Recommended Image Sizes

- **Profile Photo**: 500x500px (square, will be displayed as circular)
- **Other Images**: Optimize for web (compress to reduce file size)

## Supported Formats

- JPEG (.jpg, .jpeg)
- PNG (.png)
- GIF (.gif)
- WebP (.webp)
